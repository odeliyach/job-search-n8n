# N8N Job Search System Usage Guide

## Introduction
Welcome to the N8N Job Search System! This guide provides you with a step-by-step instruction on how to use the system effectively.

## Step 1: Accessing the N8N Job Search Workflow
1. Log in to your N8N instance.
2. Find the Job Search workflow that has been set up for you in the workflows list.

## Step 2: Pasting Job URLs
1. Locate the input field designated for job URLs in the workflow.
2. Copy the job URLs from your preferred job listing site.
3. Paste the copied URLs into the input field of the N8N Job Search workflow.
4. Ensure that each URL is on a new line, if the field allows multiple entries.

## Step 3: Executing the Workflow
1. After entering the URLs, click on the 'Execute' button to run the workflow.
2. Wait for the process to complete; this may take a few moments depending on the number of URLs and the response time of the job listing site.

## Step 4: Reviewing the Outputs
1. Once executed, navigate to the output section of the workflow.
2. You should see the following outputs:
   - Job Title
   - Company Name
   - Location
   - Job Description
   - Apply Link
3. Review the retrieved information for each job URL you pasted in.

## Conclusion
You are now ready to use the N8N Job Search System! By following these steps, you will be able to find job opportunities quickly and efficiently. Don't hesitate to experiment with different job listing sites to broaden your search.

Happy job hunting!
